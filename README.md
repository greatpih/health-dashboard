# 건강·약 트렌드 대시보드

대장님 개인용으로 건강/약 관련 기사와 공지를 추려서 보는 첫 버전입니다.

## 열어보는 방법

가장 쉬운 방법:
1. Finder에서 `health-dashboard/index.html` 파일을 더블클릭
2. 브라우저에서 바로 보기

또는 터미널에서 이 폴더로 이동 후 간단한 서버 실행:

```bash
cd health-dashboard
python3 -m http.server 8000
```

그다음 브라우저에서 아래 주소 열기:
- http://localhost:8000
